# 🚀 Laboratório de Redes - Guia Completo de Instalação

Bem-vindo ao guia completo de instalação e configuração de um laboratório de redes profissional utilizando VMware, EVE-NG, MikroTik, e sistemas Linux.

## 📋 Visão Geral do Laboratório

Este laboratório permite simular ambientes de rede complexos para:

- Treinamento em configuração de roteadores e switches
- Testes de topologias de rede
- Aprendizado de protocolos de roteamento
- Simulação de cenários reais de infraestrutura

## 🛠️ Componentes do Laboratório

| Componente             | Descrição                           | Status |
| ---------------------- | ----------------------------------- | ------ |
| **VMware Workstation** | Plataforma de virtualização base    | ✅     |
| **EVE-NG**             | Emulador de ambientes virtuais      | ✅     |
| **MikroTik RouterOS**  | Sistema operacional para roteadores | ✅     |
| **Servidor Linux**     | Servidor para serviços de rede      | ✅     |
| **Cliente Linux**      | Estação de trabalho para testes     | ✅     |

## 🎯 Pré-requisitos

- Windows 10/11 (64-bit) ou Linux
- Processador com suporte a virtualização (Intel VT-x/AMD-V)
- Mínimo 8GB RAM (recomendado 16GB+)
- 100GB de espaço em disco disponível
- Conexão com internet para downloads
